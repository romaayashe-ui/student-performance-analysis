Student Performance Analysis Dashboard

Project Overview
This project focuses on analyzing student academic performance using Excel. The goal is to transform raw educational data into clear, actionable insights through data cleaning, analysis, and interactive dashboard visualization to support data-driven academic evaluation.
Problem Statement
Educational institutions often struggle to effectively track student performance due to scattered or unstructured data. This project addresses that challenge by organizing and analyzing student grades and attendance data to identify performance trends and areas requiring academic support.

Dataset
Contains student grades, attendance records, and performance-related attributes
Raw data required cleaning and restructuring before analysis
Data was prepared to ensure accuracy, consistency, and usability

Tools & Technologies
Microsoft Excel
Power Query
Pivot Tables
Data Visualization
Dashboard Design

Methodology
Imported raw student data into Excel
Cleaned and transformed data using Power Query
Validated data and handled missing or inconsistent values
Built Pivot Tables for performance analysis
Designed KPIs to measure academic performance and attendance
Created an interactive dashboard to visualize trends and insights

Key Insights
Identified performance trends across different students
Analyzed the relationship between attendance and academic results
Highlighted students and areas requiring academic intervention
Dashboard Features
Interactive charts for grades and attendance
KPI indicators for overall performance tracking
Easy-to-read visual layout for quick decision-making

How to Use
Download the Excel file from the repository
Open the file using Microsoft Excel
Use slicers and filters to explore student performance data
Review dashboards and KPIs for insights

Future Improvements
Add predictive analysis for student performance
Integrate automated data refresh
Expand analysis to include behavioral or demographic factors
